# Smart Task & Habit Tracker

CSC480 Software Engineering Team Project

## Project Overview

The Smart Task & Habit Tracker is a full-stack MERN application designed to help users manage tasks, track habits, monitor productivity, and receive simple AI-based suggestions over time.

The project is being developed using Agile Scrum methodology with weekly sprints, Jira sprint planning, GitHub version control, and collaborative team development practices.

---

## Features

### Current Sprint 1 Features
- User registration API
- User login API
- JWT authentication
- bcrypt password hashing
- MongoDB database integration
- MERN project structure
- React frontend initialization
- Bootstrap responsive landing page

### Planned Features
- Task CRUD operations
- Habit tracking and streaks
- Dashboard analytics
- AI productivity suggestions
- User profile management
- Responsive mobile-friendly UI
